### JSON이란?
※ JSON
- JavaScript에서 객체를 만들 때 사용하는 표현식

### JSON API
- JSON.parse('문자열')
    - 인자로 전달된 문자열을 자바스크립트의 데이터로 변환한다.
- JSON.stringify('변수명')
    - 인자로 전달된 자바스크립트의 데이터를 문자열로 변환한다.
